Batch barcode label generator

A free tool for generating barcode labels in batches — built for inventory tagging, device serial numbers, IMEI tracking, and warehouse labeling. Scan or type each item's code, add your own custom label text underneath, and export everything to a print-ready PDF sized to your exact label dimensions. Works entirely in your browser — no account, no install, no data leaves your device.

Scan or type items one at a time below. Each gets its own barcode, the raw text under it, and a space for your custom label text.

How saving / loading a queue works

▸
  • Save queue downloads everything currently listed below (every barcode value and its custom text) as a single .json file to your computer's Downloads folder. Nothing is uploaded anywhere — it just saves to your device.
  • Load queue opens a file picker so you can pick a previously saved .json file back up. If you already have items queued, it'll ask whether to add the loaded items to your current list or replace it entirely.
  • This is handy for pausing a long scanning session — save before you stop, then load that same file later to pick up right where you left off — or for splitting work between two people and merging both queues into one PDF at the end.
  • Your PDF layout settings (label size, border, etc.) are not part of the saved file — only the scanned items and custom text are. Layout settings stay as whatever is currently set on this page.
  • Closing or refreshing this page without saving will lose your queue, since nothing is stored automatically — save a file first if you need to step away.
