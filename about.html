About this tool

A free, browser-based barcode label generator built to replace the old copy-paste-into-Word workflow. Scan items, add labels, export a PDF, print and cut. That's it.

What it does

This tool lets you scan or type item codes — like IMEIs, serial numbers, SKUs, or any text string — and instantly generates a scannable barcode for each one. Below each barcode you'll see the raw code printed, and you can add your own custom label text underneath (like a device name, location, or description).

When you're done adding items, hit Export PDF and you get a print-ready document laid out in a grid, sized exactly to whatever label dimensions you set. Print it, cut it out, stick it on your items.

Everything runs entirely in your browser. No account needed, no data is sent anywhere, and no internet connection is required once the page has loaded.

Frequently asked questions

Is this really free?
Yes — fully free to use, no signup, no limits on how many barcodes you generate. The site may display ads to keep it running, but the tool itself is free.
Does it work with a barcode scanner gun?
Yes. Most USB and Bluetooth barcode scanners act like a keyboard — they type the scanned value and press Enter automatically. Just click into the "Scan or type item text" field and scan away. Each item gets added to the queue instantly.
What barcode formats are supported?
Code 128 (default — works with any combination of letters and numbers), Code 39, EAN-13, UPC-A, and ITF-14. Code 128 is the right choice for most uses like IMEIs and serial numbers.
My printed barcode won't scan. What do I do?
Try turning off the barcode border in the PDF layout settings (set it to 0) and re-exporting. A border placed too close to the bars can confuse some scanners. Also make sure you're printing at 100% scale — not "fit to page" — so the barcode dimensions are accurate.
Can I save my work and come back to it later?
Yes. Use the Save queue button to download your current list of items as a small JSON file. Later, hit Load queue to pick up right where you left off. Nothing is stored online — the file lives on your computer.
How do I fit more labels on a page?
Open the PDF layout settings panel and reduce the label width and/or height. The live counter shows how many columns and rows will fit on the page as you adjust. Reducing the gap between labels also helps squeeze in more per page.
What label sheet sizes does it support?
You can enter any custom dimensions in inches. Quick presets include 3-1/16" × 1-3/16" (a standard Avery-compatible size), 2" × 1", 1.5" × 0.5", and 1.25" × 0.4". Letter (US) and A4 page sizes are both supported.
Does it work offline or on restricted work networks?
Yes. The tool and all its libraries are bundled into a single file — no external requests are needed to generate barcodes or export PDFs. If you downloaded the offline version, it works completely without internet access.
Is my data private?
Yes. Everything you type or scan stays in your browser — no data is sent to any server. See the Privacy Policy for full details.
